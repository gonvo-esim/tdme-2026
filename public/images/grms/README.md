# GRMS Images Directory

This directory contains images for the Tiancheng 669 Series GRMS webpage.

## Recommended Image Files

Upload the following images extracted from the Tiancheng 669 PDF to this directory:

- `tiancheng-669-hero.jpg` - Hero/banner image for the page
- `topology-diagram.png` - Tiancheng 669 Series Topology Diagram
- `csi-technology.png` - CSI technology visualization
- `solution-package-1.png` - Core Basic Package diagram
- `solution-package-2.png` - Essence Container diagram
- `solution-package-3.png` - Flagship Container diagram
- `huawei-wifi7-logo.png` - Huawei Wi-Fi 7 branding
- `bittel-gateway-icon.png` - Bittel Gateway icon

## Image Usage

Images are referenced in the webpage using Next.js Image component optimized paths:
```
/images/grms/[image-name]
```

## Guidelines

- Use WebP format for better compression when possible
- Maintain aspect ratios suitable for web display
- Optimize file sizes to ensure fast page loading
- Use descriptive filenames in kebab-case
